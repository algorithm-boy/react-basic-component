# Progress

## API

| 参数(props) | 说明(specification) | 类型(type) | 默认值(defaultProps) |
| --- | --- | --- | --- |
| className(optional) | Progress的类名 (The class property for Progress) | string | - |
| style(optional) | Progress的行内样式 (The inline style for Progress) | object | {} |

## Example

